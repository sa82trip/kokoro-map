# US-14: PNG 내보내기

## 스토리 개요
마인드맵을 PNG 이미지로 내보내는 기능을 구현합니다. 다양한 출력 옵션(배경색, 여백, 해상도)을 지원하며 사용자가 원하는 형태로 이미지를 생성할 수 있도록 합니다.

## 상태
**[ ] To Do** → **[x] In Progress**

**우선순위**: 보통
**점수**: 5
**기간**: 2일 (2026-03-31 ~ 2026-04-01)

## 요구사항
- 마인드맵을 PNG 이미지로 내보내기
- 배경색 옵션 (투명/흰색/현재 테마)
- 여백(Padding) 설정
- 해상도 배율 (1x, 2x, 3x)
- 내보내기 진행 중 로딩 표시

## 기술 아키텍처
```javascript
// MindMapStore 추가 상태
{
  // ... 기존 상태
  exportConfig: {
    background: 'theme',     // 'transparent' | 'white' | 'theme'
    padding: 20,            // 여백 (px)
    scale: 2,               // 배율 (1x, 2x, 3x)
  }
}

// FileExporter 확장
exportToPNG(config) → Promise<Blob>
```
