# 마인드맵 프로그램 PRD (Product Requirements Document)

## 1. 문제 정의

### 현황
- 사용자들은 생각과 아이디어를 정리할 때 종이 노트나 단순한 텍스트 에디터를 사용함
- 기존 마인드맵 툴들은 복잡하거나, 기능이 부족하거나, 비용이 비싼 문제 존재
- 팀 협업 시 아이디어 공유와 정리가 어려움
- 발표용으로 마인드맵을 시각적으로 꾸미는 기능이 부족

### 사용자 요구사항
- 복잡한 생각을 체계적으로 정리
- 눈에 띄는 시각적 표현
- 실시간 협업 기능
- 다양한 형식으로 내보내기
- 사용하기 쉬운 UI/UX

## 2. 제품 목표

### 비전
모든 사용자가 자신의 생각을 시각적으로 정리하고, 협업하며, 생산성을 극대화할 수 있는 마인드맵 플랫폼

### 목표
1. 사용자의 창의성과 생산성 향상
2. 팀 협업 효율성 증대
3. 교육 및 비즈니스 시장 점유율 확대
4. 월간 활성 사용자(MAU) 100만 달성

## 3. 타겟 사용자

### 주요 사용자
1. **직장인 (40%)**
   - 프로젝트 기획
   - 회의록 정리
   - 아이디어 발상

2. **학생/교육자 (30%)**
   - 학습 노트 정리
   - 시험 공부
   - 강의 준비

3. **크리에이터/자영업자 (20%)**
   - 콘텐츠 기획
   - 비즈니스 모델 설계
   - 브레인스토밍

4. **개발자 (10%)**
   - 아키텍처 설계
   - 요구사항 정리
   - 코드 구조 시각화

## 4. 기능 요구사항

### 4.1 기본 기능 (Must-have)

#### 4.1.1 마인드맵 생성 및 편집
- 중심 노드 생성 및 편집
- 방사형 가지 자동 생성
- 계층 구조 관리 (깊이 제한: 10단계)
- 노드 추가/삭제/이동
- 키워드 우편 편집 시스템

#### 4.1.2 시각화 기능
- 50가지 기본 색상
- 200가지 아이콘 라이브러리
- 이미지 업로드 및 삽입
- 연결선 스타일 (직선, 곡선, 화살표)
- 폰트 크기/색상/굵기 조절

#### 4.1.3 파일 관리
- 생성된 마인드맵 저장
- 최근 문서 목록
- 검색 및 필터링
- 폴더 구조 관리

### 4.2 핵심 기능 (Should-have)

#### 4.2.1 템플릿 라이브러리
- 업무용 템플릿 (20개)
- 학습용 템플릿 (15개)
- 개인 템플릿 생성 및 공유
- 템플릿 평가 시스템

#### 4.2.2 협업 기능
- 실시간 동시 편집 (최대 10인)
- 댓글 및 언급 기능 (@)
- 작업 권한 관리 (소유자, 편집자, 뷰어)
- 활동 로그 확인

#### 4.2.3 내보내기 기능
- PDF (발표용)
- PNG/JPG (이미지)
- PowerPoint/Keynote
- Word/Google Docs
- XMind/FreeMind 파일 호환

### 4.3 고급 기능 (Could-have)

#### 4.3.1 AI 보조 기능
- 자동 아이디어 추천
- 키워드 분석 및 확장
- 내용 요약 및 정리
- 다른 문서와의 연동 추천

#### 4.3.2 발표 모드
- 노드 강조 및 숨김
- 애니메이션 효과
- 프레젠테이션 타이머
- 노트 추가 기능

#### 4.3.3 통합 기능
- Google Drive 연동
- Slack/Teams 연동
- Trello/Jira 연동
- Google Calendar 연동

### 4.4 기술적 요구사항
- 반응형 디자인 (데스크탑, 태블릿, 모바일)
- 클라우드 저장 (AES-256 암호화)
- 자동 백업 (5분 간격)
- 오프라인 작업 지원

## 5. 성능 요구사항

1. **로딩 속도**
   - 초기 로딩: 2초 이내
   - 파일 열기: 1초 이내
   - 실시간 동기화: 0.5초 이내

2. **사용자 동시 접속**
   - 무제한 동시 편집 지원
   - 실시간 충돌 방지 알고리즘

3. **저장 용량**
   - 무료 사용자: 1GB
   - 유료 사용자: 100GB

## 6. 디자인 및 UX 요구사항

### UI 원칙
- **직관성**: 눈에 보이는 기능만 제공
- **일관성**: 전체적으로 통일된 디자인 언어
- **효율성**: 최소한의 클릭으로 완성
- **접근성**: 색맹/시각 장애인용 고대비 모드

### 컬러 스키마
- 기본: 흰색 배경 + 어두운 텍스트
- 어두운 모드: 다크 그레이 배경 + 밝은 텍스트
- 브랜드 컬러: #4A90E2 (파란계) + #50E3C2 (청록계)

- 마케팅 ROI

## 9. 경쟁 우위

### 차별화 포인트
1. **사용성**: 초보자도 5분 만에 익힐 수 있는 직관적 인터페이스
2. **시각화**: 최고 수준의 디자인 품질
3. **협업**: 실시간 팀워크 최적화
4. **가격**: 합리적인 유료 모델
5. **통합**: 주요 생산성 툴과의 깊은 연동

### 경쟁사 분석
- XMind: 기능은 많지만 복잡함
- MindMeister: 웹 기반 우수하지만 디자인이 평범함
- Miro: 협업 우수하지만 마인드맵 전문성 부족
- Coggle: 사용성 좋으나 기능이 제한적

## 11. FAQ

**Q: 무료 버전과 유료 버전의 차이는?**
A: 무료 버전은 5개 마인드맵 제한, 1GB 저장 공간, 기본 템플릿 제공. 유료 버전은 무제한 마인드맵, 100GB 저장, 전체 템플릿, AI 기능 등 제공.

**Q: 오프라인에서 사용 가능한가요?**
A: 웹 버전은 인터넷 연결 필요. 데스크탑 앱은 오프라인 작업 가능하나 동기화 시 연결 필요.

**Q: 여러 언어를 지원하나요?**
A: 현재 한국어와 영어 지원. 추후 일본어, 중국어, 스페인어 추가 예정.
