# KOKORO MAP - 마인드맵 앱

> 사용자 친화적인 마인드맵 애플리케이션

## 프로젝트 개요

이 프로젝트는 PRD 문서를 기반으로 한 마인드맵 앱 개발입니다. React 기반의 웹 애플리케이션으로, 다양한 플랫폼에서 사용할 수 있도록 설계되었습니다.

## 주요 기능

- ✅ 마인드맵 생성 및 편집
- 🎨 노드 추가/삭제/편집
- 📱 반응형 디자인 (모바일/데스크톱)
- 🔍 모바일 줌 컨트롤
- 🌐 빠른 로딩 성능

## 기술 스택

- **Frontend**: React, Vite
- **Styling**: CSS, CSS Modules
- **Build**: Vite
- **Deploy**: Vercel

## 시작하기

```bash
npm install
npm run dev
```

## 빌드 및 배포

```bash
# 빌드
npm run build

# 배포
vercel --prod
```
